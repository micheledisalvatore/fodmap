## Commands

Install dependencies: `$ yarn install`

Run dev mode `$ yarn start`

Build project `$ yarn build`

Deploy project on S3: `$ yarn run publish`

Check the on line project: http://fodmap-es.s3-website.us-east-2.amazonaws.com/
